# LNB, 우측 Info 레이아웃 구현
## 기능 
- 좌측 햄버거 메뉴 구현                  
- 우측 info 메뉴 구현                  
- contents 영역에서 info 클릭시 해당 내용과 관련된 내용 우측 info 메뉴 영역 노출               
- 데이터 관리 용이하도록 Vuex 사용                        
- 아이콘 사용           
- splitpanes 을 사용하여 화면 분할 구현          
- 30%, 50%, 70% 버튼 클릭시 화변 비율 변경            
- 드래그로 화변 비율 변경
