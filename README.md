# LNB, 우측 Info 레이아웃 구현
## 기능 
- 좌측 햄버거 메뉴 구현                  
- 우측 info 메뉴 구현                  
- contents 영역에서 info 클릭시 해당 내용과 관련된 내용 우측 info 메뉴 영역 노출               
- 데이터 관리 용이하도록 Vuex 사용                  
- 아이콘 사용
